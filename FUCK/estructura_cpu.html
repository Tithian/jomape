Estructura general de un microprocesador

El Contador de Programa (PC) es un registro que indica donde se encuentra la próxima instrucción a ejecutar en la RAM. Una operación de lectura carga la instrucción en el Registro de Instrucciones (Rl). Cada instrucción tiene dos campos:

  • Código de operación (CO), expresa el tipo de operación a realizar. Manda a la Unidad de Control las señales necesarias para ejecutar la instrucción.
  • Operandos (OPS) indica dónde se encuentran los operandos y activa los registros de esa operación.

El Banco de Registros (BR) sirve para almacenar los operandos.

El Registro de Estado es un conjunto de Biestables que indican condiciones del resultado.

El Registro Acumulador almacena el resultado de las operaciones. Puede reenviarlo al Banco de Registros para encadenar operaciones o enviarlo al Bus de Datos.

La ALU está formada por:

  • Unidad aritmética. Realiza las operaciones aritméticas básicas y suele estar formada por un sumador en C2 y, en algunos casos, por un multiplicador.
  • Unidad lógica. Realiza las operaciones booleanas de desplazamiento y/o rotación.

Ambas unidades trabajan en punto fijo. Para realizar las operaciones en punto flotante se utiliza un coprocesador aritmético.
