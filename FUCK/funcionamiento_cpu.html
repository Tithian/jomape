Funcionamiento de la Unidad Central de Proceso

Un programa a ejecutar está compuesto por un conjunto de instrucciones almacenadas en la memoria. La CPU se encarga de ejecutarlas.

Ciclo de Instrucción: Procesamiento que requiere una instrucción. Tiene dos etapas:

  • Busqueda de instrucción: es una operación común a todas las instrucciones, es la lectura de la instrucción de una posición de memoria. (También conocido como: Ciclo de Captación).
  • Ejecución de instrucción: consiste en realizar todas las operaciones de la instrucción. (También conocido como: Ciclo de Ejecución).

Al comienzo de cada ciclo de instrucción, la CPU capta una instrucción de memoria y tras almacenarla en el RI, incrementa el PC. La Unidad de Control interpreta el código y genera señales de control para ejecutarla (salvo que se indique otra cosa).

La ejecución del programa es la repetición del proceso de búsqueda y de ejecución de las instrucciones del programa realizando las siguientes etapas:

  1. La CPU lee de memoria la instrucción.
  2. Decodifica la instrucción para conocer el tipo de operación a realizar y dónde están los operandos.
  3. Se leen los operandos bien de memoria o de E/S.
  4. Ejecuta la operación.
  5. Se guardan los resultados en memoria o E/S.
  6. Se lee la siguiente instrucción.

Este proceso se repite hasta terminar el programa.
