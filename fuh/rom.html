ROM: PROM, EPROM, EEPROM, Flash Memory

Read Only Memory sobrevive al apagado al ser no volátil. necesita una pila.

PROM (Programmable ROM)

Solo se puede programar una vez convirtiéndose en una memoria ROM, permitiéndonos sólo su lectura.

EPROM (Erasable PROM)

Nos permite grabar y borrar reiteradas veces mediante radiaciones ultravioleta. Presenta el inconveniente de que se borra toda la información.

EEPROM (Electrically EPROM)

Es una memoria que puede borrar su información mediante elevadas corrientes eléctricas. Podemos borrar sólo una parte de la memoria, quedando el resto de la información intacta. Presenta el inconveniente que sólo se puede cambiar un byte de información cada vez.

Flash Memory

Es un tipo de EEPROM que se puede reprogramar en bloques. Se usa en la BIOS.

Hasta un lavavajillas podría tener una ROM
